## 后台管理 (nova-mall-admin)

### 认证与权限

- 管理员登录/注册/退出
- JWT Token 生成/刷新
- 动态资源权限加载
- 角色-菜单/资源分配

### 商品管理 (PMS)

- 商品列表/添加/编辑/删除
- 品牌管理（CRUD + Logo 上传）
- 商品分类管理（三级分类树）
- 商品属性管理（规格参数 + 销售属性）
- SKU 库存管理
- 商品审核/上下架
- 商品操作日志

### 订单管理 (OMS)

- 订单列表/详情/发货
- 退货申请处理（审核/拒绝）
- 退货原因设置
- 订单设置（超时时间等）
- 公司地址管理

### 营销管理 (SMS)

- 优惠券管理（创建/编辑/删除/领取记录）
- 秒杀活动管理（活动/场次/商品关联）
- 首页广告管理
- 首页推荐（品牌/新品/热销/专题）

### 用户管理 (UMS)

- 管理员管理（CRUD/角色分配/状态切换）
- 角色管理（CRUD/菜单分配/资源分配）
- 菜单管理（树形结构/图标/排序）
- 资源管理（CRUD/分类）
- 会员等级管理

### 内容管理 (CMS)

- 专题管理
- 优选专区

### Agent 查询接口

- 销售统计/热销商品/库存预警
- 会员统计/优惠券统计/订单状态分布
- 仪表盘数据/订单趋势

## 商城前台 (nova-mall-portal)

### 首页

- 轮播广告/品牌推荐/新品推荐
- 热销商品/秒杀活动
- 商品分类导航

### 商品

- 商品搜索（ES 全文检索）
- 商品详情（属性/规格/价格）
- 品牌专区（推荐列表/详情/商品）

### 购物车

- 加购/修改数量/删除/清空
- 促销信息展示

### 订单

- 确认订单（地址/优惠券/备注）
- 生成订单（责任链模式）
- 订单列表/详情/取消/确认收货
- 支付回调处理
- 超时订单自动取消（RabbitMQ 延迟队列）

### 售后

- 申请退货
- 售后列表/状态查询/删除

### 优惠券

- 可领取优惠券列表
- 领取优惠券
- 我的优惠券/购物车可用优惠券

### 会员

- 登录/注册/获取验证码
- 会员信息/刷新 Token
- 收货地址管理（CRUD/默认地址）
- 浏览历史/商品收藏/品牌关注
- 商品评价

### 支付

- 支付宝支付（沙箱）
- 支付结果回调

## 搜索服务 (nova-mall-search)

- 商品数据同步至 ES
- 按名称/品牌/分类全文检索
- IK 中文分词

## AI 智能客服 (nova-mall-agent)

### 客服助手（23 个工具）

- 秒杀/购物车/订单/售后/优惠券/地址/会员
- 混合分级路由（正则 + LLM）
- 代码状态机（地址修改/售后申请）
- 二次确认机制

### 后台管理助手（6 个工具）

- 销售统计/热销商品/库存预警
- 会员统计/优惠券统计/订单状态分布
- 运营日报自动聚合
- RAG 知识问答

## 前端 (nova-mall-admin-web)

- 登录/权限路由
- 商品管理（多步骤表单）
- 订单管理（状态筛选/详情/发货）
- 营销管理（优惠券/秒杀/广告/推荐）
- 权限管理（管理员/角色/菜单/资源）
- AI 客服对话窗口
- ECharts 数据可视化

## 移动端 (nova-mall-app-web)

- H5 / 微信小程序 / Android / iOS 多端发布
- 首页/分类/商品/购物车/订单/支付
- 售后/地址/优惠券/会员中心
- AI 客服对话（WebSocket）
